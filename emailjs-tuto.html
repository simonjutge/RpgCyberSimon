CYBERQUEST
Tuto EmailJS — Simon reçoit un mail à chaque inscription Guest
?
POURQUOI EMAILJS ?
Quand un Guest s'inscrit dans CyberQuest, Simon reçoit automatiquement un email avec :
le nom du guest, son mot de passe, la date/heure, et un rappel pour activer JSONBin public si besoin.
✅ Gratuit · 200 emails/mois · Sans backend · Fonctionne depuis iPhone et Quest 2
⚠️ Fonctionne avec n'importe quelle adresse email — pas besoin de Gmail.
1
CRÉER UN COMPTE EMAILJS
Ouvrir emailjs.com →
1
Clique Sign Up Free → entre ton adresse email et un mot de passe
2
Confirme l'email reçu → connecte-toi au dashboard
Utilise l'adresse email où tu veux recevoir les notifications guests (ton adresse perso suffit).
2
CONNECTER TON ADRESSE EMAIL
1
Dashboard EmailJS → menu gauche → Email Services
2
Clique Add New Service
3
Choisis ton service email :
• Gmail → connexion directe avec ton compte Google
• Outlook / Hotmail → même principe
• Personal Email (SMTP) → pour Free, Orange, SFR, etc. → entre serveur SMTP + login + mot de passe
• Yahoo Mail → connexion directe
4
Donne un nom au service (ex: cyberquest-notif) → Connect Account → autorise l'accès
5
Copie le Service ID affiché
Exemple : service_ab12cd34
Pour Free (Zimbra) ou autre SMTP :
Serveur : smtp.free.fr | Port : 587 | Ton login Free + mot de passe
3
CRÉER LE TEMPLATE EMAIL
1
Menu gauche → Email Templates → Create New Template
2
Remplis exactement comme ci-dessous (les noms entre {{ }} sont importants) :
To :{{to_email}}
Subject :🔐 CyberQuest — Guest : {{guest_name}}

Body : Bonjour Simon,

Nouvelle action Guest sur CyberQuest.

Action : {{action}}
Heure : {{time}}

Nom : {{guest_name}}
Mot de passe : {{guest_pwd}}

{{message}}

— CyberQuest
3
Clique Save → copie le Template ID
Exemple : template_xy56zw78
4
RÉCUPÉRER LA PUBLIC KEY
1
Menu gauche → Account → onglet General
2
Section Public Key → copie la valeur
Exemple : aBcDeFgHiJkLmNoP12
5
COLLER LES VALEURS DANS index.html
Dans index.html sur GitHub, cherche et remplace ces 4 lignes :
var EMAILJS_PUBLIC_KEY = 'aBcDeFgHiJkLmNoP12';
var EMAILJS_SERVICE_ID = 'service_ab12cd34';
var EMAILJS_TEMPLATE_ID = 'template_xy56zw78';
var SIMON_EMAIL = 'ton@adresse.fr';
✅ C'est tout ! Chaque inscription Guest envoie un email automatiquement à Simon.
SIMON_EMAIL = l'adresse où Simon veut recevoir les mails (peut être différente de celle du compte EmailJS).
6
TESTER
1
Ouvre CyberQuest iPhone → chip Guest → onglet Créer mon compte
2
Entre un nom + mot de passe → valide
3
Simon doit recevoir l'email en moins de 30 secondes
4
Si rien → vérifie les spams + que les 4 valeurs sont collées correctement
Limite gratuite : 200 emails/mois.
Si dépassé, les notifications s'arrêtent silencieusement — l'app continue de fonctionner normalement.
+
GÉRER LES GUESTS (panel Admin)
Une fois un Guest inscrit, Simon ouvre le panel Admin (bouton Gérer) pour :
→
Suspendre / Restaurer — bloque la connexion du guest sans supprimer son compte
→
Zones — limite les zones accessibles (ex: uniquement Python Dungeon et Network Fortress)
→
XP max — plafonne l'XP gagnable (ex: 300 XP max)
→
Bonus ON/OFF — active ou désactive les quêtes bonus et scénarios pour ce guest
→
Supprimer — supprime définitivement le compte
Le mot de passe choisi par le Guest est toujours visible dans le panel Admin.
